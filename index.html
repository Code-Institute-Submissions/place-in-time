Welcome to Place in Time
A few things to note before you get started...
  • This site requires access to your device location in order to display results in your proximity. Please ensure all location services are active on your device before continuing.
  • Please ensure all adblockers and tracking blockers are disabled for this site, as third party service providers use this info to deliver best results.
  • Remember to reload the page when you find yourself in a new area. You will update interesting locations for that area by doing so.
How to use:
  1. Select a location by clicking on a map marker.
  2. Each marker will have an overview paragraph, providing a brief introduction to the location.
  3. Many locations will have an added history section which you can expand by clicking on the box which displays the message "Tap here to show history".
  4. If you are ready to be transported back in time, simply press the button below!
